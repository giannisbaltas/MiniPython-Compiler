Εργασία


Σκοπός της εργασίας είναι η υλοποίηση ενός μεταγλωττιστή για τη γλώσσα miniPython. Η εργασία αυτή αποτελεί το 50% του τελικού βαθμού στο μάθημα.

Γλώσσα miniPython.

Η γλώσσα miniPython. είναι ένα υποσύνολο της Python και επομένως το νόημα ενός προγράμματος σε miniPython. είναι ίδιο με το νόημα του ίδιου προγράμματος σε Python. 

Θα χρησιμοποιήσετε το εργαλείο SableCC που θα σας διευκολύνει στην κατασκευή του μεταγλωττιστή.


Επιπλέον βοήθεια μπορείτε να βρείτε επίσης στο βιβλίο Modern Compiler Implementation in Java του Andrew W. Appel

Υποστηριζόμενες εντολές

Η γλώσσα miniPython υποστηρίζει τα εξής:

  1. Ανάθεση ακεραίου, δεκαδικού και συμβολοσειράς σε μεταβλητή
  2. Ανάθεση λίστας (list) σε μεταβλητή
  3. Δήλωση συναρτήσεων με απλά ορίσματα, ορίσματα με προεπιλεγμένες τιμές (default values)
  4. Οι εντολές if, print, while και for 
  5. Κλήση συναρτήσεων
  6. Σχόλια μιας γραμμής

Δομή Εργασίας

Η εργασία θα χωριστεί σε 2 Φάσεις:

Φάση Α

  1. Λεκτική ανάλυση της γλώσσας miniPython.
  2. Συντακτική ανάλυση της γλώσσας miniPython.

Φάση Β

  1. Aφηρημένα συντακτικά δέντρα της γλώσσας miniPython.
  2. Πίνακας Συμβόλων - Σημασιολογική Ανάλυση της γλώσσας miniPython.

Τα δύο πρώτα και τα δύο τελευταία τμήματα θα εξεταστούν ενιαία. Επομένως θα υπάρχουν συνολικά 2 φάσεις (2 παραδοτέα).

Συμβουλές

Για να μπορέσετε τις παραπάνω εντολές που εμφανίζονται σε κάθε φάση θα πρέπει να βεβαιωθείτε ότι τα προγράμματα java κα javac βρίσκονται στη μεταβλητή περιβάλλοντος PATH και ότι τις εκτελείτε από φάκελο που δουλεύετε την εργασία σας ή αλλιώς τον αρχικό φάκελο (έτσι θα αναφερόμαστε παρακάτω).
