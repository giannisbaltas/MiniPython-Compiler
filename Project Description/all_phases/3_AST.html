Αφηρημένο Συντακτικό Δέντρο

Σκοπός της άσκησης

Η επέκταση του αρχείου ορισμού γραμματικής που προέκυψε από τις δύο προηγούμενες ασκήσεις, έτσι ώστε να αυτοματοποιηθεί η κατασκευή αφηρηρημένων συντακτικών δέντρων κάνοντας χρήση του εργαλείου SableCC.

Παρεχόμενος κώδικας

Θα πρέπει να βασιστείτε πάνω στο αρχείο ορισμού γραμματικής που έχετε δημιουργήσει στις προηγούμενες φάσεις και να το επεκτείνετε. Και εδώ χρειάζεται το SableCC. Επίσης σας δίνονται τα αρχεία ASTPrinter.java και ASTTest1.java. Θα σας βοηθήσουν στην επιβεβαίωση της ορθότητας του αφηρημένου συντακτικού δέντρου μέσω της οπτικοποίησης του ΑΣΔ.

Δημιουργία αφηρημένου συντακτικού δέντρου

Ο στόχος της κατασκευής ενός αφηρημένου συντακτικού δέντρου είναι να εγκαταλείψουμε όσα στοιχεία, όσους κόμβους, στο συγκεκριμένο συντακτικό δέντρο, μας είναι άχρηστα για τα επόμενα στάδια της μεταγλώττισης. Έτσι, μέχρι τη συντακτική ανάλυση χρειαζόμασταν συγκεκριμένα tokens για να ξεχωρίζουμε ποιον κανόνα από όλους θέλουμε να εφαρμόσουμε ή χρειαζόμασταν διάφορα επίπεδα κανόνων για να δηλώσουμε προτεραιότητες και να επιλύσουμε ενδεχόμενες συγκρούσεις. Όταν έχουν αποφασιστεί όλα αυτά από το συντακτικό αναλυτή, δεν είναι πλέον σκόπιμο να κρατάμε μεγάλο όγκο πληροφορίας.

Θα πρέπει να προσθέσετε στο αρχείο ορισμού γραμματικής, από κάτω από τα υπόλοιπα τμήματα που αναφέρθηκαν στην λεκτική ανάλυση, το τμήμα:

Abstract Syntax Tree // Δήλωση του ΑΣΔ
...

 αλλάζοντας ταυτόχρονα το τμήμα Productions από την προηγούμενη φάση ώστε να συμβαδίζει με το τμήμα Abstract Syntax Tree. Επεξήγηση για τις μετατροπές που θα σας βοηθήσουν στην δημιουργία των αφηρημένων συντακτικών δέντρων υπάρχει στο link AST transformations in SableCC 3.0. Σας δίνονται επίσης οδηγίες στην ελληνική γλώσσα για τους μετασχηματισμούς ΑΣΔ στο εργαλείο SableCC.

Έστω ότι το αρχείο σας ονομάζεται minipython.grammar . Τότε εκτελέστε:
sablecc minipython.grammar
javac ASTTest1.java
java ASTTest1 example.py

Θα παρατηρήσετε ότι η έξοδος πλέον περιέχει το αφηρημένο συντακτικό δέντρο.
