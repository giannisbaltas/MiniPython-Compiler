Λεκτική ανάλυση

Σκοπός της άσκησης

Η κατασκευή ενός λεκτικού αναλυτή για την miniPython κάνοντας χρήση του εργαλείου SableCC.

Πέρα από τα βοηθητικά αρχεία και σημειώσεις που παρέχει το site, μπορείτε να διαβάσετε και το manual του SableCC.

Απαραίτητα Αρχεία

Σας δίνεται το sablecc.jar το οποίο πρέπει να κατεβάσετε τοπικά και να το αντιγράψετε σε ένα φάκελο lib ο οποίος θα βρίσκεται στο φάκελο που δουλεύετε την εργασία σας. Δηλαδή αν π.χ ο φάκελος που δουλεύετε την εργασία σας είναι ο “compiler” τότε πρέπει να δημιουργήσετε έναν φάκελο με όνομα “lib” μέσα στον φάκελο “compiler” και να αντιγράψετε εκεί το αρχείο sablecc.jar. Επιπλέον αντιγράψετε το αρχείο sablecc.bat (για Windows) στον φάκελο που δουλεύετε την εργασία.

Το αρχείο sablecc.bat διευκολύνει την εκτέλεση του sablecc και περιέχει την εντολή:
java -jar lib\sablecc.jar %1 %2 %3 %4 %5 %6 %7 %8 %9

Επίσης σας δίνεται το αρχείο LexerTest1.java το οποίο περιέχει μία main μέθοδο η οποία καλεί το λεκτικό αναλυτή και απλά εμφανίζει στην οθόνη τα αποτελέσματα της εκτέλεσης.

Για όσους δουλεύετε την εργασία σε περιβάλλον Linux ή IRIX, θα πρέπει να χρησιμοποιήσετε το αρχείο sablecc αντί του sablecc.bat. Σε αυτήν την περίπτωση σιγουρευτείτε ότι έχετε προσθέσει δικαιώματα εκτέλεσης στο αρχείο sablecc που κατεβάσατε (chmod +x sablecc).

Δημιουργία λεκτικού αναλυτή

Αυτό που εσείς πρέπει να κάνετε είναι η δημιουργία ενός αρχείου ορισμού γραμματικής. Το αρχείο αυτό μπορεί να περιέχει τα εξής τμήματα για την φάση αυτή:

Package minipython; // Δήλωση του πακέτου, συνεπώς και φακέλου, στο οποίο θα τοποθετηθούν τα παραγόμενα αρχεία.

Helpers // Βοηθητικά στοιχεία για τα παρακάτω
...

States // Δήλωση των καταστάσεων, στις οποίες μπορεί να βρίσκεται το αυτόματο της λεκτικής ανάλυσης
...

Tokens // Οι λεκτικές μονάδες της λεκτικής ανάλυσης
...

H δήλωση πακέτου πρέπει να είναι οπωσδήποτε η παραπάνω (minipython) διότι το όνομα του φακέλου χρησιμοποιείται στα imports του LexerTest1.java. Σας δίνονται σύντομες οδηγίες για την λεκτική ανάλυση.

Έστω ότι το αρχείο σας ονομάζεται minipython.grammar. Τότε βρισκόμενοι στον φάκελο που δουλεύετε την εργασία εκτελέστε:
sablecc minipython.grammar

Μετά την εκτέλεση θα πρέπει να έχει δημιουργηθεί ένας κατάλογος με όνομα minipython ο οποίος περιέχει τους εξής τέσσερις καταλόγους: analysis, lexer, node και parser. Αμέσως μετά εκτελέστε από τον ίδιο κατάλογο:
javac LexerTest1.java
java LexerTest1 minipythonexample.py

Το αποτέλεσμα θα πρέπει να είναι η εκτύπωση όλων των tokens χωρισμένα από κενά.
