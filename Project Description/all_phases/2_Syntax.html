Συντακτική ανάλυση

Σκοπός της άσκησης

Η κατασκευή ενός συντακτικού αναλυτή για την miniPython κάνοντας χρήση του εργαλείου SableCC. Ουσιαστικά πρόκειται για μία επέκταση της προηγούμενης άσκησης του λεκτικού αναλυτή.

Απαραίτητα Αρχεία

Θα πρέπει να βασιστείτε πάνω στο αρχείο ορισμού γραμματικής που έχετε δημιουργήσει στην προηγούμενη φάση και να το επεκτείνετε. Και εδώ χρειάζεται το SableCC. Επίσης σας δίνεται το αρχείο ParserTest1.java το οποίο περιέχει μία main μέθοδο η οποία καλεί τον συντακτικό αναλυτή και απλά εμφανίζει στην οθόνη τα αποτελέσματα της εκτέλεσης.

Δημιουργία συντακτικού αναλυτή

Θα πρέπει να προστεθούν στο αρχείο ορισμού γραμματικής, από κάτω από τα υπόλοιπα τμήματα που αναφέρθηκαν στην λεκτική ανάλυση, τα εξής επιπλέον τμήματα:

Ignored Tokens // Tokens που αγνοούνται, τυπικά τα σχόλια και τα κενά
...

Productions // Οι κανόνες της γραμματικής της γλώσσας
...

Στο τμήμα Productions γράφουμε τους κανόνες που αποτελούν τη γραμματική που παράγει την επιθυμητή γλώσσα. Πρέπει να επισημανθεί ότι η γραμματική που δίνεται είναι διφορούμενη (ambiguous). Αυτό σημαίνει ότι αν προσπαθήσετε να την μετατρέψετε άμεσα σε αρχείο ορισμού γραμματικής τη SableCC θα υπάρξουν αρκετές reduce/reduce ή/και shift/reduce συγκρούσεις. Επομένως είναι απαραίτητη επαναδιατύπωση ή/και πρόσθεση κάποιων κανόνων έτσι ώστε επιλυθούν θέματα προτεραιότητας μεταξύ των κανόνων όπου αυτό χρειάζεται. Σε καμία περίπτωση όμως δεν πρέπει να αλλάξει η σημασιολογία της γλώσσας. Δίνεται ένα παραδείγμα αντιμετώπισης προτεραιότητας. Δίνονται επίσης σύντομες οδηγίες για την συντακτική ανάλυση στο SableCC.

Έστω ότι το αρχείο σας ονομάζεται minipython.grammar . Τότε εκτελέστε στον αρχικό κατάλογο:
sablecc minipython.grammar
javac ParserTest1.java
java ParserTest1 minipythonexample.py

Και εδώ το σωστό αποτέλεσμα θα πρέπει να είναι η εκτύπωση όλων των tokens χωρισμένα από κενά.
