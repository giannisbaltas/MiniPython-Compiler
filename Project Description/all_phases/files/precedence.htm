Προτεραιότητες τελεστών

postfix operators [ ] (params)
unary operators !
multiplicative *
additive + -
relational <
logical AND &&
assignment =

Το (params) υπονοεί παραμέτρους σε κλήση μεθόδου, και όχι παρενθέσεις γύρω από μια έκφραση που προφανώς έχουν την ανώτατη προτεραιότητα

Η σειρά με την οποία εφαρμόζονται οι τελεστές, όταν βρίσκονται στο ίδιο επίπεδο προτεραιότητας είναι από τα αριστερά προς τα δεξιά. Δεν υπάρχει δυνατότητα στη minipython να βρίσκονται στο ίδιο statement δύο εκχωρήσεις, ώστε δε μας ενδιαφέρει η προσεταιριστικότητα της εκχώρησης '='.
