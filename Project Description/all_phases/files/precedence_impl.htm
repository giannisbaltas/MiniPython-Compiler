Υλοποίηση προτεραιότηας τελεστών

Ακολουθεί ένα πρακτικό παράδειγμα για την εφαρμογή της προτεραιότητας των τελεστών σε ένα αρχείο γραμματικής για το SableCC. Πρόκειται για την απόδοση υψηλότερης προτεραιότητας στον πολλαπλασιαμό έναντι της πρόσθεσης. Υλοποιείται ακόμα η από τα αριστερά προς τα δεξιά προσεταριστικότητα των πράξεων.

Μια γενική τακτική, είναι να υπάρχει ένα επίπεδο κανόνα.

  
 multiplication = {something} something 
   		| {multiplication} multiplication times something
 		;
   
 addition = 
  	{multiplication} multiplication |
	{addition} addition plus multiplication|
 	{subtraction} addition minus multiplication;
